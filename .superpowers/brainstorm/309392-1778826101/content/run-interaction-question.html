What happens when you trigger a run?

The run already executes async in the background — question is what you see while it's happening

A

Live streaming output

Log lines appear in real-time as the monitor runs (Server-Sent Events). You watch it like a terminal. Page auto-refreshes the run history when done.

B

Trigger and poll

Click "Run now", page polls every second until the run appears in the DB, then shows the result. Simpler, no streaming infrastructure needed.
